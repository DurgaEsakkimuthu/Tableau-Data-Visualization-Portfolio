# Tableau-Data-Visualization-Portfolio
A collection of Tableau visualization techniques including aggregations, calculated fields, joins, sorting, and multiple chart types.

## Project Overview
This project demonstrates various Tableau data visualization techniques and analytical features. The goal is to explore how different chart types and calculations can be used to analyze data effectively.

## Topics Covered

### 1. Tableau Aggregations
- Sum
- Minimum
- Maximum
- Average
- Count
- Count Distinct
- Median
- Variance
- Standard Deviation

### 2. Quick Table Calculations
- Percentage of Total
- Running Total
- Difference
- Rank
- Moving Average

### 3. Tableau Calculated Fields
- Operators
- IF–ELSE conditions
- String functions
- Ranking functions

### 4. Data Preparation
- Joining tables
- Sorting techniques

### 5. Visualization Types

The following charts were implemented:

- Bar Chart
- Line Chart
- Pie Chart
- Scatter Plot
- Pareto Chart
- Motion Chart
- Lollipop Chart
- Gantt Chart
- Cross Tab
- Bump Chart
- Bubble Chart
- Waterfall Chart
- Area Chart
- Heat Map
- Tree Map
